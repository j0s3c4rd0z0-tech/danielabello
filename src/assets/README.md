# 📸 Carpeta de Assets — DB Salon

Coloca aquí todos los archivos de imagen y media del proyecto.

## Archivos esperados:

| Archivo | Descripción |
|---------|-------------|
| `logo_db.png` | Logo principal DB Salon (fondo transparente, PNG) |
| `logo_db_white.png` | Variante blanca del logo (opcional) |
| `hero-bg.jpg` | Imagen de fondo del Hero (mínimo 1920×1080px) |
| `about-photo.jpg` | Foto para la sección Nosotros |
| `service-*.jpg` | Fotos de cada servicio |
| `product-*.jpg` | Fotos de productos de la tienda |
| `favicon.png` | Ícono del sitio (32×32 o 64×64px) |

## Notas:
- Para cambiar cualquier imagen, actualiza la ruta en `src/content/SITE_CONTENT.js`
- Usa imágenes de alta calidad para mantener el aspecto premium de la marca
- Formatos recomendados: `.jpg` para fotos, `.png` para logos con transparencia, `.webp` para mejor rendimiento
